**Бесплатное Windows-приложение для безопасного сетевого туннелирования**

## 📖 О приложении

**NeonTunnel** — современное решение для безопасного и приватного доступа в интернет, разработанное с учетом всех требований к конфиденциальности в цифровую эпоху.

### 🎯 Наша миссия

Мы создали NeonTunnel, чтобы дать пользователям полный контроль над их интернет-трафиком и защитить их цифровую свободу. В мире, где каждый клик может быть отслежен, мы предлагаем надежный щит между вами и сетью.

## 🛠 Возможности

### 🔐 Безопасность и конфиденциальность

**Защита от слежки**
NeonTunnel создает защищенный канал между вашим устройством и интернетом, предотвращая анализ вашего трафика третьими сторонами.

**Отсутствие следов**
Приложение не сохраняет историю ваших действий, не создает лог-файлы и не оставляет следов использования на вашем компьютере.

**Антианализ**
Встроенная система защиты от попыток изучения работы приложения через отладчики и инструменты анализа.

### 🌐 Универсальность

**Четыре режима работы** — каждый режим создан для конкретных задач:

| Режим | Описание | Когда использовать |
|-------|----------|-------------------|
| **DPI Only** | Легкая защита без туннеля | Базовая защита, повседневное использование |
| **Tunnel Only** | Полный VPN для максимальной приватности | Скрыть весь трафик, доступ к ресурсам |
| **Combined** | Комбинация для сложных сетей | Максимальная надежность |
| **Game Tunnel** | Оптимизация для онлайн-игр | Минимальный пинг в играх |

## 🚀 Быстрый старт

### 📥 Установка

1. Запустите установщик `NeonTunnel-Setup.exe`
2. Следуйте инструкциям мастера установки
3. После установки приложение запустится автоматически

### 🎯 Первый запуск

При первом запуске NeonTunnel откроется в режиме **DPI Only** — это легкая защита, которая не требует дополнительных настроек.

### 🔗 Добавление серверов

**Способ 1: Импорт по ссылке**
1. Скопируйте ссылку на подписку или сервер в буфер обмена
2. Нажмите кнопку **＋** в правом верхнем углу
3. Приложение автоматически добавит все серверы из ссылки

**Способ 2: Импорт из файла**
1. Нажмите меню **☰** (три полоски)
2. Выберите **Import from file**
3. Укажите файл с серверами

**Способ 3: Сканирование QR-кода**
1. Нажмите кнопку **📷** (камера)
2. Наведите камеру на QR-код с сервером

### ⚡ Активация защиты

1. Выберите нужный сервер из списка (кликните на карточку)
2. Выберите режим работы в выпадающем списке
3. Нажмите кнопку **ACTIVATE SYSTEM**
4. Для отключения нажмите **DISCONNECT**

## 📋 Режимы работы

### 🛡 DPI Only (По умолчанию)

**Когда использовать:**
- Для базовой защиты без влияния на скорость
- Когда нужен только обход ограничений
- Для повседневного использования

**Что делает:**
- ✅ Защищает от анализа трафика
- ✅ Минимальное влияние на производительность
- ✅ Не создает виртуальных сетевых интерфейсов

---

### 🔒 Tunnel Only

**Когда использовать:**
- Для максимальной приватности
- Когда нужно скрыть весь трафик
- Для доступа к ресурсам с ограничениями

**Что делает:**
- ✅ Направляет весь трафик через защищенный туннель
- ✅ Скрывает реальные IP-адреса
- ✅ Обеспечивает полную конфиденциальность без каких-либо утечек

---

### 🔄 Combined

**Когда использовать:**
- В сложных сетевых условиях
- Когда другие режимы не работают
- Для максимальной надежности

**Что делает:**
- ✅ Комбинирует несколько уровней защиты
- ✅ Автоматически адаптируется к условиям сети
- ✅ Подходит для продвинутых пользователей

---

### 🎮 Game Tunnel

**Когда использовать:**
- Для онлайн-игр
- Для стриминга с минимальной задержкой
- Когда важен каждый миллисекунда пинга

**Что делает:**
- ✅ Игровой трафик идет напрямую (минимальный пинг)
- ✅ Остальной трафик — через защищенный туннель
- ✅ Автоматически определяет игровые процессы

> ⚠️ **Важно:** Этот режим не поддерживает все типы серверов. Если увидите предупреждение о несовместимости — переключитесь на другой режим.

## ⚙️ Настройки и персонализация

### 🎨 Смена цветовой темы

1. Нажмите меню **☰**
2. Выберите **Settings**
3. В разделе **Color Theme** выберите готовую тему или создайте свою
4. Нажмите **Apply**

Тема применится сразу и сохранится для будущих запусков.

---

### 🖥 Управление серверами

**Редактирование сервера:**
1. Наведите курсор на карточку сервера
2. Нажмите на иконку **⚙️** (настройки)
3. Измените параметры и нажмите **Save**

**Удаление сервера:**
1. Наведите курсор на карточку сервера
2. Нажмите на иконку **🗑️** (удалить)
3. Подтвердите удаление

💡 **Совет:** Приложение автоматически проверяет доступность серверов. Зеленый индикатор показывает время отклика.

## ❓ Часто задаваемые вопросы (FAQ)

### 📌 Общие вопросы

**Q: NeonTunnel бесплатный?**
A: Да, приложение полностью бесплатно.

**Q: Нужно ли создавать аккаунт?**
A: Нет, регистрация не требуется. Приложение работает сразу после установки.

**Q: Какие операционные системы поддерживаются?**
A: На данный момент поддерживается Windows 10 и Windows 11.

**Q: Нужны ли права администратора?**
A: Да, для работы с сетевыми настройками требуются права администратора. Приложение запросит их при первом запуске.

---

### 🔐 Безопасность

**Q: Собирает ли NeonTunnel мои данные?**
A: Нет. Мы не собираем, не храним и не передаем никакие данные о вашей активности.

**Q: Создает ли приложение логи?**
A: Нет. NeonTunnel спроектирован так, чтобы не оставлять следов использования на вашем устройстве.

**Q: Можно ли доверять приложению?**
A: Да. Приложение не содержит вредоносного кода, не отправляет данные на внешние серверы и не устанавливает скрытые компоненты.

**Q: Что такое "антианализ" и зачем он нужен?**
A: Это система защиты от попыток изучения работы приложения через отладчики. Это нужно для защиты от вредоносного ПО, которое может пытаться анализировать работу защитных программ.

---

### ⚡ Производительность

**Q: Влияет ли NeonTunnel на скорость интернета?**
A: В режиме DPI Only влияние минимально. В режимах с туннелем скорость зависит от выбранного сервера и его удаленности.

**Q: Увеличивает ли приложение пинг в играх?**
A: В режиме Game Tunnel игровой трафик идет напрямую, что обеспечивает минимально возможный пинг.

**Q: Можно ли использовать NeonTunnel для стриминга?**
A: Да, приложение оптимизировано для работы с видеопотоками и обеспечивает стабильное соединение.

---

### 🎯 Использование

**Q: Что делать, если сервер не работает?**
A: Попробуйте:
- Выбрать другой сервер из списка
- Переключить режим работы
- Проверить интернет-соединение

**Q: Почему я вижу предупреждение о несовместимости режима?**
A: Не все режимы поддерживают все типы серверов. Приложение автоматически предложит совместимый режим.

**Q: Можно ли использовать несколько серверов одновременно?**
A: Нет, активен только один выбранный сервер. Но вы можете быстро переключаться между ними.

---

### 🔧 Технические вопросы

**Q: Какие протоколы поддерживаются?**
A: Приложение работает с современными защищенными протоколами. Конкретный протокол определяется сервером.

**Q: Шифруется ли мой трафик?**
A: Да, весь трафик через туннель шифруется современными алгоритмами.

**Q: Можно ли использовать NeonTunnel с другими VPN?**
A: Технически возможно, но не рекомендуется. Это может привести к конфликтам и нестабильной работе.

**Q: Что делать при ошибке подключения?**
A: Приложение автоматически пытается переподключиться до 5 раз. Если это не помогает — выберите другой сервер или режим.

---

### 🎮 Режим Game Tunnel

**Q: Почему Game Tunnel не работает с некоторыми серверами?**
A: Этот режим требует специфической поддержки от сервера. Если видите предупреждение — используйте режим Combined или Tunnel Only.

**Q: Какие игры поддерживаются?**
A: Поддерживаются практически все популярные онлайн-игры: Steam, Epic Games, Riot Games, Ubisoft, EA и другие.

**Q: Нужно ли настраивать Game Tunnel для каждой игры?**
A: Нет, приложение автоматически определяет игровые процессы и применяет оптимальные настройки.

## 🛠 Решение проблем

### ❌ Приложение не запускается

**Проблема:** После установки приложение не открывается.

**Решение:**
1. Запустите приложение от имени администратора
2. Проверьте, что Windows Defender или антивирус не блокирует приложение
3. Переустановите приложение

---

### 🔌 Не удается подключиться

**Проблема:** При нажатии ACTIVATE SYSTEM ничего не происходит или появляется ошибка.

**Решение:**
1. Проверьте интернет-соединение
2. Выберите другой сервер
3. Попробуйте режим DPI Only
4. Перезапустите приложение

---

### 📈 Высокий пинг в играх

**Проблема:** В режиме Game Tunnel пинг выше ожидаемого.

**Решение:**
1. Убедитесь, что выбран ближайший сервер
2. Проверьте, что игра определена как игровой процесс
3. Попробуйте режим DPI Only для минимального пинга

---

### 🔄 Приложение работает нестабильно

**Проблема:** Соединение периодически разрывается.

**Решение:**
1. Выберите другой сервер (возможно, текущий перегружен)
2. Попробуйте режим Combined для большей стабильности
3. Проверьте стабильность вашего интернет-соединения
