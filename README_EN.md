## 🚀 Quick Start

### 🎯 First Launch
On first launch, NeonTunnel will open in **DPI Only mode** — this is lightweight protection that requires no additional configuration.

### 🔗 Adding Servers

**Method 1: Import via Link**
1. Copy the subscription or server link to clipboard
2. Click the **＋** button in the top right corner
3. The application will automatically add all servers from the link

**Method 2: Import from File**
1. Click the **☰** menu (three lines)
2. Select **Import from file**
3. Specify the file with servers

**Method 3: QR Code Scanning**
1. Click the **📷** (camera) button
2. Point the camera at the server QR code

### ⚡ Activating Protection
1. Select the desired server from the list (click on the card)
2. Select the operating mode from the dropdown list
3. Click the **ACTIVATE SYSTEM** button
4. To disconnect, click **DISCONNECT**

## ❓ Frequently Asked Questions (FAQ)

### 📌 General Questions

**Q: Is NeonTunnel free?**  
A: Yes, the application is completely free.

**Q: Do I need to create an account?**  
A: No, registration is not required. The application works immediately after installation.

**Q: Which operating systems are supported?**  
A: Currently supports Windows 10 and Windows 11.

**Q: Are administrator rights required?**  
A: Yes, administrator rights are required to work with network settings. The application will request them on first launch.

---

### 🔐 Security

**Q: Does NeonTunnel collect my data?**  
A: No. We do not collect, store, or transmit any data about your activity.

**Q: Does the application create logs?**  
A: No. NeonTunnel is designed to leave no traces of usage on your device.

**Q: Can I trust the application?**  
A: Yes. The application contains no malicious code, does not send data to external servers, and does not install hidden components.

**Q: What is "anti-analysis" and why is it needed?**  
A: This is a protection system against attempts to study the application's operation through debuggers. This is needed to protect against malware that may try to analyze the operation of protection programs.

---

### ⚡ Performance

**Q: Does NeonTunnel affect internet speed?**  
A: In DPI Only mode, the impact is minimal. In tunnel modes, speed depends on the selected server and its distance.

**Q: Does the application increase ping in games?**  
A: In Game Tunnel mode, gaming traffic goes directly, providing the minimum possible ping.

**Q: Can I use NeonTunnel for streaming?**  
A: Yes, the application is optimized for working with video streams and provides a stable connection.

---

### 🎯 Usage

**Q: What to do if a server doesn't work?**  
A: Try:
- Select another server from the list
- Switch the operating mode
- Check your internet connection

**Q: Why do I see a mode incompatibility warning?**  
A: Not all modes support all server types. The application will automatically suggest a compatible mode.

**Q: Can I use multiple servers simultaneously?**  
A: No, only one selected server is active. But you can quickly switch between them.

---

### 🔧 Technical Questions

**Q: Which protocols are supported?**  
A: The application works with modern secure protocols. The specific protocol is determined by the server.

**Q: Is my traffic encrypted?**  
A: Yes, all traffic through the tunnel is encrypted with modern algorithms.

**Q: Can I use NeonTunnel with other VPNs?**  
A: Technically possible, but not recommended. This may lead to conflicts and unstable operation.

**Q: What to do in case of connection error?**  
A: The application automatically attempts to reconnect up to 5 times. If this doesn't help — select another server or mode.

---

### 🎮 Game Tunnel Mode

**Q: Why doesn't Game Tunnel work with some servers?**  
A: This mode requires specific server support. If you see a warning, use Combined or Tunnel Only mode.

**Q: Which games are supported?**  
A: Virtually all popular online games are supported: Steam, Epic Games, Riot Games, Ubisoft, EA, and others.

**Q: Do I need to configure Game Tunnel for each game?**  
A: No, the application automatically detects game processes and applies optimal settings.

## 🛠 Troubleshooting

### ❌ Application Won't Launch
**Problem:** After installation, the application doesn't open.

**Solution:**
1. Run the application as administrator
2. Check that Windows Defender or antivirus is not blocking the application
3. Reinstall the application

---

### 🔌 Unable to Connect
**Problem:** When clicking ACTIVATE SYSTEM, nothing happens or an error appears.

**Solution:**
1. Check your internet connection
2. Select another server
3. Try DPI Only mode
4. Restart the application

---

### 📈 High Ping in Games
**Problem:** In Game Tunnel mode, ping is higher than expected.

**Solution:**
1. Make sure the nearest server is selected
2. Check that the game is detected as a game process
3. Try DPI Only mode for minimum ping

---

### 🔄 Application is Unstable
**Problem:** Connection periodically drops.

**Solution:**
1. Select another server (the current one may be overloaded)
2. Try Combined mode for greater stability
3. Check the stability of your internet connection
